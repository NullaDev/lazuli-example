## 创建一个可用的窗口
创建一个最基本的窗口，有按钮进入菜单、游戏和退出按钮，并且显示静态的背景画面和文本

## 完成基本的游戏逻辑
基本的游戏逻辑包括：

- 从菜单进入游戏后，在底部显示文本，并且有静态的背景画面
- 鼠标点击画面任意处或按空格键、Enter键可以切换文本和背景画面
- 在游戏中能够显示一个或几个选项来使游戏进入不同方向的发展
- 在剧情结束后退出游戏的选项
